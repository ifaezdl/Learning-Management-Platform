# EduCore LMS

A modern Learning Management System (LMS) built with Next.js, NestJS, and PostgreSQL.

## Features

- User Authentication
- Student Dashboard
- Teacher Dashboard
- Course Management
- Lesson Management
- Enrollment System
- Payment Integration
- Progress Tracking
