# Roles

## Student

Permissions:

- Register
- Login
- View courses
- Purchase courses
- Access enrolled courses
- View lessons
- Track progress

---

## Teacher

Permissions:

- Register
- Login
- Create courses
- Edit courses
- Create lessons
- Upload videos
- View enrolled students

---

## Admin

Permissions:

- Manage users
- Manage courses
- Manage categories
- View system statistics
